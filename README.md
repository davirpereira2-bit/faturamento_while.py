# faturamento_while.py

### Descrição
Módulo de fechamento de caixa desenvolvido para gerenciar o faturamento total de uma sessão de vendas. Focado em acumuladores numéricos e controle de fluxo contínuo.

### Funcionalidades
- **Acumulação de Receita:** Soma dinâmica de cada venda realizada ao saldo total do sistema.
- **Loop de Operação:** Mantém o caixa aberto para múltiplas transações consecutivas.
- **Relatório de Fechamento:** Gera um resumo financeiro detalhado ao encerrar as atividades.

### Conceitos Aplicados
- **Operadores de Atribuição Composta:** Uso de `+=` para persistência de soma em variáveis.
- **Finalização de Fluxo:** Implementação de critério de parada pelo usuário (`break`).
